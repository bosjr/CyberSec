## Objectif
Ce dépôt sert de point d’entrée unique vers :

des projets pédagogiques
des laboratoires (labs)
des write-ups CTF
des supports de cours
...

Chaque élément est isolé dans un dépôt dédié afin de garantir :

une meilleure organisation
un versioning indépendant
une réutilisation simplifiée

## Utilisation pédagogique

Ce hub est utilisé dans le cadre de formations en cybersécurité pour :

guider les étudiants
centraliser les accès aux TP
structurer les parcours d’apprentissage
